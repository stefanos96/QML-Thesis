# Quantum Machine Learning Thesis Code

This repository contains the Qiskit implementations and experimental code developed for my diploma thesis on Quantum Machine Learning with limited qubits.

The notebook includes quantum circuits, data encoding methods, training procedures, gradient-based optimization, and experimental results used in the thesis.

## Contents

- `QML_thesis_code.ipynb`: Main Jupyter Notebook containing the complete implementation and experiments.

## Technologies

- Python
- Qiskit
- NumPy
- Matplotlib
- scikit-learn

## Thesis Context

The code supports the experimental part of a diploma thesis focused on Quantum Machine Learning models with a limited number of qubits.
